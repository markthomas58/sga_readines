# sga_readiness

SGA readiness assessment tool.

## Requirements

- Python 3.12+

## Setup

```bash
pip install hatch
```

## Development

```bash
# Start a shell in the Hatch environment
hatch shell

# Run tests
hatch test

# Build the package
hatch build
```

## Project Structure

```
src/
  sga_readiness/       # Main source code
tests/                 # Test files
pyproject.toml         # Project config & dependencies
```
